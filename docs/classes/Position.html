    hash

    • get hash(): string

      A Zobrist hash of the position as a 16-character hex string. Computed once and cached. Two positions with the same hash are considered identical (with negligible collision probability).

      Returns string

      Remarks

      The hash algorithm and format are not stable across major versions. Do not persist hashes across version upgrades.
